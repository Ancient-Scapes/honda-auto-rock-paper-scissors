## Usage

- clone
- yarn install
- make .env

```env
ACCOUNT_ID='Twitter_ID'
ACCOUNT_PASSWORD='Twitter_Password'
```

- node index.js

## 指定したアカウントでペプシ公式宛に本田とじゃんけんツイートが送信されます

## 即ペプシ公式からリプライが飛んでくるのであとは手動で確認してください
